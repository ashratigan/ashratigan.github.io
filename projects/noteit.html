About

This app was designed as a fun way for someone to brainstorm and save ideas. I'm always thinking of new ideas for projects, but if I don't have somewhere to write them down, I'll forget them. Now, I have a place to save my ideas!

Before coding, I created an ERD, wireframe, and user stories. I also planned the React components that I would use, and the general structure of the app. In addition, I created a schedule for myself with a rough estimate of how long each feature would take, and I set aside time for teaching myself more about React.

This is a full stack, single page application. I began by creating the back end using Ruby on Rails. The front end was made with React. I used Axios to connect the React front end to the Rails back end.

In action

The first feature I completed was create new note. I added a button to the top of the page, and with each click a new sticky note appears! When a new note appears the editing form displays, so a user can add a title and contents right away. The note can be edited later by double clicking on it.

I utilized the React-Draggble package to allow users to drag sticky notes anywhere on their screen. A single click allows a user to drag the note anywhere on the screen, while a double click allows a user to edit the contents of the note. See demonstration below.

I wanted this app to be a way for people to brainstorm and write down ideas all in one, so I added a fun addition where a user who needs inspiration can click a button and a random quote, trivia fact, and writing prompt is displayed. The quotes come from talaikis.com, the trivia facts come from opentdb.com, and the writing prompt is from ineedaprompt.com
